CSS Selectors and Specificity

A Cheat Sheet

CSS Selectors

Type of Selector Example HTML Example CSS Selector Description
Tag <p>...</p> p Selects all elements with a given tag name.
Class <h3 class="brand">...</h3> brand Selects all elements with a given class name. Classes can be reused for more than one HTML element.
ID <h2 id="more-info">...</h2> more-info Selects an element with a given ID name. IDs can only be used once for any given HTML element.
Chaining Selectors <h1 class="special">...</h1> h1.special Selects all elements with a given combination of selectors, by chaining on one or more selectors with no space in between their names. In this example, the selector does not select an h1 element if it does not have the class attribute special.
Nested Elements <ul class="main-list">
  <li> ... </li>
  <li> ... </li>
  <li> ... </li>
</ul> 		.main-list li Selects all elements of a given selector that are nested inside elements of another selector, separate by a space. In this example, all li tags nested inside the HTML element with class name main-list are selected.
Multiple Selectors <h1>...</h1>
...
<div class="more-info">...</div>
 h1, .more-info Selects multiple selectors independently in order to apply the same style to all of them. In this example, HTML elements with h1 tags and HTML elements with more-info class names are both selected.
